# CAT Tracker 📚

A minimal and aesthetic productivity dashboard built for CAT preparation.

Track study hours, mock performance, topic progress, goals, revision, and overall preparation in one place.

---

## ✨ Features

- Study streak & progress tracking
- Mock test analysis
- Topic-wise preparation tracking
- Goal management
- Revision planning
- Countdown timer
- Motivational dashboard
- Clean pastel UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
